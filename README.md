# Requires:
* An osuAPIKey (save in ./Dependencies/osuAPIKey.json)
* An DiscordBotToken (save in ./Dependencies/botToken.json)
* An RiotAPIKey (save in ./Dependencies/RiotAPIKey.json)

## Required npm modules
```sh
npm install
fs
node-osu
require-dir
discord.js
nhentai-api-js
leagueapiwrapper
```
